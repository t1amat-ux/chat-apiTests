	API для управления чатами и сообщениями с использованием FastAPI, PostgreSQL и Docker.

	Функциональность

- Создание чатов
- Отправка сообщений в чаты
- Получение чата с последними сообщениями
- Удаление чатов (каскадное удаление сообщений)
- Валидация данных (длина, пустые строки, тримминг)
- Пагинация сообщений
- Логирование
- Полное покрытие тестами

	Быстрый старт

	Предварительные требования
- Docker и Docker Compose
- Python 3.11+ (для локальной разработки)

	Запуск с Docker Compose

```bash
# Клонируйте репозиторий
git clone <repository-url>
cd chat-api

# Запустите приложение
docker-compose up -d

# Приложение будет доступно по адресу:
# http://localhost:8000

Swagger UI: http://localhost:8000/docs

ReDoc: http://localhost:8000/redoc
